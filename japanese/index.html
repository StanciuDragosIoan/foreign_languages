Japanese Writing

There are 3 alphabetes in Japanese: Hiragana, Katakana and Kanji.

Hiragana is the basic Japanese alphabet. Contrary to the English alphabet however, each character represents one syllable sound.

The second script used in Japanese is Katakana. Each Katakana character represents the same syllable as its Hiragana counterpart. However, it is written a bit differently and is used mostly for foreign words.

Hiragana

Hiragana is the basic Japanese alphabet. Contrary to the English alphabet however, each character represents one syllable sound (a phoneme). It comprises the below table:


The Hiragana Vowels (あ,い,う,え,お)

あ (A)

い (i)

う (U)

え (E)

お (O)

Basic words with the first 5 vowels

あい (Love)

い (Stomach)

いいえ (No)

うえ (Up/Above)

あお (Blue)

The Hiragana K Column (か き く け こ)

か (KA)

*with the ゛ (dakuten) dyacritic this becomes が (GA)

き (KI)

*with the ゛ (dakuten) dyacritic this becomes ぎ (GI)

く (KU)

*with the ゛ (dakuten) dyacritic this becomes ぐ (GU)

け (KE)

*with the ゛ (dakuten) dyacritic this becomes げ (GE)

こ (KO)

*with the ゛ (dakuten) dyacritic this becomes ご (GO)

The Hiragana S Column (さ し す せ そ)

さ (SA)

*with the ゛ (dakuten) dyacritic this becomes ざ (ZA)

し (SHI)

*with the ゛ (dakuten) dyacritic this becomes じ (DJI)

す (SU)

*with the ゛ (dakuten) dyacritic this becomes ず (ZU)

せ (SE)

*with the ゛ (dakuten) dyacritic this becomes ぜ (ZE)

そ (SO)

*with the ゛ (dakuten) dyacritic this becomes ぞ (ZO)

*the letter そ has a second variant (which is ソ )
